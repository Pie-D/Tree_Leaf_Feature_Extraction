## Trích rút đặc trưng ảnh lá cây dựa trên
- Đặc trưng về màu sắc:
  - Sử dụng kỹ thuật Color Histogram trong hệ màu HSV
- Đặc trưng về hình dạng:
  - Sử dụng kỹ thuật HOG (histogram of oriented gradients)
